# 🇧🇷 Português

## 🚀 Sobre o Projeto

**Glitch Run** é um jogo de plataforma 2D desenvolvido na **Unity 6** como projeto acadêmico da disciplina de Desenvolvimento de Jogos.

O jogador controla um fragmento de código preso em um ambiente digital corrompido, utilizando falhas do sistema ("glitches") para manipular a realidade, superar obstáculos e escapar de uma simulação em colapso.

O projeto foi desenvolvido com foco na aplicação prática de conceitos de Game Design, programação, integração de sistemas e desenvolvimento de jogos utilizando a Unity.

---

## 🎯 Principais Mecânicas

* Plataforma 2D com movimentação precisa
* Progressão em quatro fases
* Materialização temporária de plataformas
* Manipulação de elementos do ambiente através de glitches
* Obstáculos e desafios com dificuldade progressiva
* Interface inspirada em sistemas digitais

---

## 🛠 Tecnologias Utilizadas

* Unity 6
* C#
* Git
* GitHub

# 🇺🇸 English

## 🚀 About the Project

**Glitch Run** is a 2D platformer developed with **Unity 6** as an academic project for a Game Development course.

Players control a fragment of code trapped inside a corrupted digital world, using system glitches to manipulate reality, overcome obstacles and escape from a collapsing simulation.

The project focuses on applying practical concepts of Game Design, gameplay programming, system integration and game development using Unity.

---

## 🎯 Main Features

* Precision 2D platforming
* Four progressively challenging levels
* Temporary platform materialization
* Reality manipulation through glitch mechanics
* Environmental hazards and platforming challenges
* Digital-inspired interface and visual identity
