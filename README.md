## 🇧🇷 Português

### 🚀 Sobre o Projeto

Glitch Run é um jogo de plataforma 2D desenvolvido na Unity como projeto acadêmico da disciplina de desenvolvimento de jogos.

O jogador explora um mundo digital corrompido onde a realidade pode ser manipulada através de mecânicas de glitch, permitindo criar plataformas temporárias, atravessar obstáculos e resolver desafios.

---

### 🎯 Principais Mecânicas

* Materialização de plataformas
* Manipulação da realidade
* Progressão por fases
* Obstáculos e desafios ambientais
* Elementos visuais inspirados em falhas digitais

---

### 🛠 Tecnologias

* Unity 6
* C#
* Git
* GitHub

---

### 👥 Equipe

Projeto desenvolvido por alunos da graduação em Ciência da Computação.

## 🇺🇸 English

### 🚀 About the Project

Glitch Run is a 2D platformer developed in Unity as an academic game development project.

Players explore a corrupted digital world where reality can be manipulated through glitch mechanics, allowing them to create temporary platforms, overcome obstacles, and solve environmental challenges.

---

### 🎯 Main Features

* Platform materialization
* Reality manipulation
* Level progression
* Environmental obstacles
* Digital glitch-inspired visuals

---

### 🛠 Technologies

* Unity 6
* C#
* Git
* GitHub

---

### 👥 Team

Developed by Computer Science students.
